2.	How to insert border in Excel with Format Cells dialog?
ans:	To insert borders in Excel using the Format Cells dialog:
			a.Select the cells or range you want to format.
			b.Right-click and choose "Format Cells."
			c.Go to the "Border" tab.
			d.Choose the desired border style under the "Presets" or customize it using the buttons below.
			e.Click "OK" to apply the borders.

		b.Go to the "Home" tab on the Ribbon.

		d.Choose "Currency" from the list.

6.	How do you use text commands in Excel?
ans:	The TEXT function in Excel converts numbers from the numerical format to text format by using formatting codes.
		Syntax:- 
			=TEXT(value, format_text)
		Example:-
			=TEXT(C9,”dd mmmm, yyyy”) 
			Converts the numerical Date 20-04-2000 into the text format 20April, 2000
