2. If you set a row height or column width to 0 (zero), what happens to the row and column?
ans:
	Setting a row height or column width to 0 in Excel hides the respective row or column. While data remains intact, it becomes invisible on the spreadsheet. This technique is useful for temporary hiding without deleting content, and to reveal, the height or width must be reset.

5. How to hide rows containing blank cells?
ans:
	To hide rows with blank cells in Excel, select the desired rows, go to "Find & Select" in the "Home" tab, choose "Go To Special", pick "Blanks," and click "OK." Right-click on the selected rows and choose "Hide".

6. What are the steps to hide the duplicate values using conditional formatting in excel?
ans:
	Firstly select the data -> Select conditional formatting -> Click on Highlight cell rules -> Click on Duplicate Value -> End .
